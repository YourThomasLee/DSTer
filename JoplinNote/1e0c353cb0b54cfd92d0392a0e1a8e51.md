查看dump文件： 
	```bash
	> gdb bin_file core_file
	> bt #查看所有堆栈
	```
开启调试日志记录：`set trace-commands on`

设置断点：`break file_name:line_number if condition`
删除断点：`delete break_point_id`
关闭显示长度限制：`set print elements 0`
